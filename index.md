## 📖 Troubleshooting common issues

If you encounter problems, check these solutions.

*   The agent does not see the plugin: Close the agent software and open it again. Ensure you placed the plugin files in the correct directory.
*   File will not run: Check that you have an active internet connection. Windows sometimes blocks files from unknown sources. Right-click the file, choose "Properties," and check "Unblock" near the bottom of the window.
*   Performance issues: Close other demanding applications while the agent works. Creative processing requires memory.
*   Connection timeouts: Verify your firewall settings. Sometimes security software prevents local data communication between applications.
